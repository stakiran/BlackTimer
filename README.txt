==================
    BlackTimer
==================

Windows で使える超シンプルなタイマーツール。


■インストール方法
(1) 以下 URL から Zip ファイルをダウンロードします。
→ https://github.com/stakiran/BlackTimer/archive/master.zip

(2) Zip ファイルを展開後、適当なフォルダに配置します。


■使い方
blacktimer.bat が本体です。
これを起動するとタイマーをセットできます。

起動すると質問が二点ほど投げられますので、
適宜入力してください。

  「[Step 1/2] 何分後にリマインドしますか？」
   → これは何分後にメッセージを表示するかを指定します。
      0 ～ n です。
      3 なら 3 分後ですし、60 なら 60 分後です。

  「[Step 2/2] リマインド時のメッセージは？(空欄可)」
   → これは表示メッセージを指定します。
      「カップラーメンできた」のようにわかりやすく書いても良いですし、
      何も書かなくても構いません。

いくつか例を挙げます。

  [例1] 3 分後にカップラーメンができたことをリマインドしたい
   → Step 1/2 は 3、Step 2/2 は「カップラーメン」

  [例2] 25 分後にとにかく気付かせてほしい
   → Step 1/2 は 25、Step 2/2 は未入力（あるいは好きなメッセージを）

  [例3] どのように表示されるかをテストしたい
   → Step 1/2 は未入力、Step 2/2 も未入力

■(補足1) リマインドについて
リマインドとは指定時間経過後に指定メッセージを表示することです。
Remind(思い出させる)、という意味の言葉です。

BlackTimer では以下方法でリマインドを行います。

  ・最大化したメモ帳を表示する
  ・そのメモ帳の内容に、指定したメッセージが書いてある

つまり「指定時間になると、指定メッセージが書かれたメモ帳を起動する」というのが
BlackTimer の仕組みです。


■(補足2) リリマインドについて
リリマインドとは「容赦なきリマインド(Relentless Remind)」を意味します。
一度リマインドしてもユーザーが気付くとは限らないので、
気付かせることを重視して何度も執拗にリマインドすることを指します。
詳細は以下記事をご覧ください。

  容赦なきリマインド『リリマインド(Relentless Remind)』について - タスク管理に恋してる
  → https://ilovetaskmanagement.hatenablog.com/entry/2018/11/08/215531

BlackTimer はリリマインドを搭載しています。
すべてのメモ帳ウィンドウを閉じるまで、30 秒ごとにメモ帳を起動します。
うっとうしく感じられるかもしれませんが、リマインドに気付かせるための策です。
